创建项目时的提示词，我觉得这个提示词格式很适合作为模板去跟ai沟通，来跟ai确认项目边界。
「一切以AI记录灵感为核心目标，有你觉得值得优化的可以提出来」

构建一个自动总结黏贴内容的灵感剪切板应用，这个应用界面应是一个按周组织的手账式界面：用户粘贴截图或者添加卡片黏贴文本 → AI 自动总结提炼关键词 → 图片对应要点标签排列，所有内容按周历排列和储存。

技术栈
● 前端：React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui + Framer Motion
● 后端：Node.js + Express（暂定，因为我不知道python这种简单代码能不能实现或者有哪些其他实现方式）
● 数据库：PostgreSQL（或者sqllit？） + Drizzle ORM（这个效果是啥）
● AI：自己配置apikey

核心功能

1. 周视图布局
● 固定三行结构：
  ○ 按周来分，当前周内整体是一个类似drawio的，可以鼠标按住拖拽移动来腾出空白的黏贴版
● 顶部导航：
  ○ 切换上一周 / 下一周
  ○ 显示当前周数与日期范围

2. 图片卡片
● 图片或者文本信息放置当前周内
● 图片展示样式：
  ○ 宝丽来 / 拍立得风格
  ○ 随机装饰（胶带 / 图钉 / 回形针 / 和纸胶带）
● 文本展示样式：
  ○ 随机颜色卡片
  ○ 随机装饰（胶带 / 图钉 / 回形针 / 和纸胶带）
● 上传完成后：
  ○ 自动调用大模型
  ○ 生成该图片或文本对应的提炼总结关键词

3. 术语标签（Terminology）
● 每张图片或文本卡片右侧显示关键词标签
● 默认展示：
  ○ 第一个关键词 +「N」
● 悬停：
  ○ 展开完整关键词列表
● 交互：
  ○ 单个标签悬停显示删除按钮（X），可删除该关键词

4. 设计约束
● 整体色调：温暖琥珀色系
● 字体和设计风格不要看起来很正式
